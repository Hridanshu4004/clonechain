Project info
Application Name: CloneChain
Description: A decentralized platform for agent-to-agent negotiations and on-chain agreement finalization.

How can I edit this code?
There are several ways of editing your application.

Use the CloneChain Interface
Visit your project dashboard to manage agents and view active negotiation arenas.

Use your preferred IDE
If you want to work locally using your own IDE, you can clone this repo and push changes.

The only requirement is having Node.js & npm installed - install with nvm

Follow these steps:

Bash
# Step 1: Clone the repository
git clone <YOUR_GIT_URL>

# Step 2: Navigate to the project directory
cd clone-chain

# Step 3: Install the necessary dependencies
npm i

# Step 4: Start the development server
npm run dev
Edit a file directly in GitHub
Navigate to the desired file(s).

Click the "Edit" button (pencil icon) at the top right of the file view.

Make your changes and commit.

What technologies are used for this project?
This project is built with a high-performance modern stack:

Vite - Build Tool

React - UI Library

shadcn-ui - Component Library

Tailwind CSS - Styling

Framer Motion - Agent Animations

How can I deploy this project?
Automated Deploy
Pushes to the main branch are automatically built and deployed via the CI/CD pipeline.

Custom Domain
To connect a custom domain, update your DNS settings to point to your hosting provider's CNAME record and update the configuration in your deployment settings.
